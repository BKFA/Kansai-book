CKEditor Sample — XML Templates

In this sample the XmlTemplates plugin is available. It allows to load templates defined in a XML file like FCKeditor.

The plugin extends the behavior of the templates_files option, so those urls that are prefixed with "xml:" are parsed according to the XML Templates rules for FCKEditor, it's possible to specify multiple templates by using the "templateName" attribute in the <Templates> node.
